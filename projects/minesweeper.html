MineSweeperDN

MineSweeperDN is an iOS application that puts a modern twist on the classic game of minesweeper. I co-created and designed this app during the winter of 2015 as my final project for the App Development Training Program at Cornell, a one course introductory class to iOS programming taught by members of the CUAppDev core team. Download the app from iTunes and submit a review or check out the source code on github!



This success of this project marked the onset of my future pursuits in mobile app development. It lead me to join CUAppDev's core team as an iOS developer where I contribute to highly sophisticated apps and expand my programming knowledge on a versatile team. I've been able to gain practical experience on this project team where designers, front-end developers, and backend engineers collaborate to push efficient code and create coherent iOS applications.
